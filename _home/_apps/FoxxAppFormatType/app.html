FOXX OS App Format / App Builder Assister

FOXX OS uses a half-compressed app format that consists of the following:

  1. Main App file (app.html)
  2. App Hub Icon (.jpg)
  3. Browser Icon (favicon.ico)
  4. (if using nedia) Any Media used by the app(inside a _media folder)

Main App File (app.html)

The main app file, or app.html, is like a manifest. It is used by the App Hub to open the app.

The app.html CANNOT contain any languages besides HTML, CSS, JavaScript, or Vue. The OS is not Configured for others than those.

app.html files can be created by copying any fully-independent app.html files, then modified to create your desired app.

app.html files CAN contain tags to embed other websites. Nothing non-child-friendly, Nothing requiring pay to use, unless allowed by the creator.


App Hub Icon (.jpg)

The App must also have a App Icon for use in the App Hub.

Format:

  • (appnamehere) is the Apps name
  • .jpg MUST be the file extension
  • The image MUST be 64x64

Simple to follow. Its best if you make your own Icon in Paint, or find one then resize it.


Browser Icon (favicon.ico)

Simple as making an image, then resizing it in paint to 16x16.

Save as .png, then manually change the file extension to .ico


(If using Media) Any media files used by your app inside a _media folder

The header is self-explanitory.
